# AI面试官系统 - 启动指南

## ✅ 系统已成功部署

### 📁 项目结构
```
ai_interview_houxuanren/
├── index.html                    # 主入口页面
├── pages/                        # 页面目录
│   ├── candidate_portal.html    # 候选人入口
│   ├── select_role.html         # 岗位选择
│   ├── upload_resume.html       # 简历上传
│   ├── choose_interviewer.html  # 面试官选择
│   └── simulate_training.html   # 模拟训练
├── public/
│   ├── styles/
│   │   └── main.css             # 主样式文件
│   └── scripts/

│       ├── actions.js           # 动作处理
│       ├── state.js             # 状态管理
│       └── questions.js         # 题库
├── HOW_TO_DEMO.md               # 演示脚本
└── 启动指南.md                  # 本文件
```

### 🚀 启动方式

#### 方法1：Python HTTP服务器（推荐）
```bash
# 进入项目目录
cd "/e/【AI训练】/ai_interview_houxuanren"

# 启动服务器
python -m http.server 8000

# 在浏览器中打开
http://localhost:8000/
```

#### 方法2：直接打开
直接双击 `index.html` 文件（可能会有跨域限制）

### 🌐 访问地址

- **主入口**: http://localhost:8000/
- **候选人流程**: http://localhost:8000/pages/candidate_portal.html

### 🎯 完整流程测试

1. **入口页面** → 点击"🚀 开始训练"
2. **岗位选择** → 选择一个岗位（如"高级前端开发工程师"）
3. **简历上传** → 上传简历文件（PDF/DOCX/MD）
4. **面试官选择** → 选择面试官类型
5. **模拟训练** → 回答面试问题
6. **查看报告** → 查看面试评估结果

### ✨ 系统特性

- ✅ 纯前端实现，无需后端
- ✅ 使用 localStorage 持久化数据
- ✅ 响应式设计，支持移动端
- ✅ 事件委托架构，高性能
- ✅ 状态管理模式
- ✅ 支持离线使用

### 🛠️ 技术栈

- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **架构**: 事件委托 + 状态管理
- **存储**: localStorage
- **服务器**: Python http.server

### 📝 注意事项

1. 确保浏览器支持 localStorage
2. 建议使用 Chrome/Edge/Firefox 最新版本
3. 服务器运行期间不要关闭终端
4. 所有数据存储在浏览器本地

### 🐛 故障排除

**问题**: CSS或JS文件加载失败  
**解决**: 确保目录结构正确，使用HTTP服务器而非直接打开文件

**问题**: 页面跳转404  
**解决**: 检查文件名是否正确，确保在 `pages/` 目录下

**问题**: localStorage错误  
**解决**: 清除浏览器缓存，允许localStorage权限
